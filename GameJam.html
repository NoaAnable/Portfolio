Participated in the 2025 Game Maker Tool Kit 4-day game jam, where we made a rogue-lite shooter based on the given theme "Loop". I was responsible for all visual aspects, including animation, sprites, backgrounds, and overall visual design. Working on a heavily time-restricted project was challenging, but it helped us determine the core element essential to the gameplay loop and visual appeal.
